# 相場の手入力メモ（楽天でヒットしない品種の補完）

楽天市場に出品がない希少種・ハウスネーム系は、自動収集では相場が「データなし」になります。
店頭・イベント・SNS・オークション等で見かけた価格を**手入力メモ**として記録できます。

※メルカリ・ヤフオク等の自動収集は規約上行いません（この手入力が正規の補完手段です）。

## 記入のしかた

Claudeに次の形式で伝えれば、`data\price-history.js` に追記されます。

```
【相場メモ】
品種：バカラ
ステージ：成魚（卵／針子／成魚のいずれか）
価格：ペア 15,000円
見た場所：江戸オクのInstagram出品（URL）
見た日：2026-07-20
備考：ペア価格。単品換算不明
```

複数品種・複数ステージをまとめて伝えてもかまいません。

## データ上の扱い

- `source` は「手入力メモ」として記録され、楽天収集分と区別されます
- 楽天と同じく「品種ID × 調査日 × 情報源」のスナップショット型で履歴に残ります
- 価格が1点しかない場合は 最安＝平均＝最高 として記録し、件数は1になります
- 出所URLと見た日を必ずセットで残します（出典必須の設計ルール）
